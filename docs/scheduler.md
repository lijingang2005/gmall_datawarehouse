# 调度流程

## 概述

本项目使用 **Apache DolphinScheduler** 作为全流程工作流调度平台，将数据采集 → ODS 同步 → 逐层计算 → 数据导出串联为一条 DAG（有向无环图）工作流，实现每日自动化执行。

## 四、每日调度参数

### 4.1 调度时间

- **触发时间**：每日凌晨 01:00
- **依赖模式**：当天数据依赖前一天的 DWD/DWS 计算完成

### 4.2 日期传参

```bash
# 默认取 T-1 日期（昨天日期）
if [ -n "$1" ]; then
    do_date=$1
else
    do_date=$(date -d "-1 day" +%F)
fi
```

## 五、调度入口脚本

### 5.1 每日离线调度 (`offline_daily.sh`)

```bash
#!/bin/bash
# 每日离线全流程调度

DO_DATE=${1:-$(date -d "-1 day" +%F)}

echo "========== 开始执行 ${DO_DATE} 离线数仓调度 =========="

# 1. ODS 层
echo ">>> 执行 ODS 层..."
bash script/collect/hdfs_to_ods_log.sh ${DO_DATE}
bash script/collect/hdfs_to_ods_db.sh ${DO_DATE}
bash script/ods/ods_load.sh ${DO_DATE}

# 2. DIM 层
echo ">>> 执行 DIM 层..."
bash script/dim/ods_to_dim.sh ${DO_DATE}

# 3. DWD 层
echo ">>> 执行 DWD 层..."
bash script/dwd/ods_to_dwd.sh ${DO_DATE}

# 4. DWS 层
echo ">>> 执行 DWS 层..."
bash script/dws/dwd_to_dws_1d.sh ${DO_DATE}
bash script/dws/dws_1d_to_dws_nd.sh ${DO_DATE}
bash script/dws/dws_1d_to_dws_td.sh ${DO_DATE}

# 5. ADS 层
echo ">>> 执行 ADS 层..."
bash script/ads/dws_to_ads.sh ${DO_DATE}

# 6. 导出到 MySQL
echo ">>> 导出 ADS 数据到 MySQL..."
bash script/export/hdfs_to_mysql.sh ${DO_DATE}

echo "========== ${DO_DATE} 离线数仓调度完成 =========="
```

### 5.2 调度策略配置

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 失败重试次数 | 3 | 任务失败后自动重试 |
| 重试间隔 | 5 分钟 | 每次重试等待时间 |
| 超时告警 | 2 小时 | 单个任务超时自动告警 |
| 并行度 | Worker 数×2 | 最大并行任务实例数 |

## 六、监控与告警

- **任务状态监控**：DolphinScheduler UI 可视化查看每个任务实例的实时状态
- **告警通知**：任务失败/超时时通过 Alert 服务发送通知（支持邮件、钉钉等）
- **日志查看**：每个 Worker 节点均提供日志服务，可直接查看任务执行日志
